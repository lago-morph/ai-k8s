# Tutorial 01: Create an S3 Bucket with Crossplane

## Overview

This tutorial provides a hands-on introduction to using mk8 and Crossplane to provision AWS infrastructure. You'll learn the fundamental workflow of infrastructure-as-code by creating an S3 bucket on AWS using a local Kubernetes cluster.

## What's Covered

- Installing and configuring mk8
- Creating a local bootstrap cluster
- Installing Crossplane and the AWS Provider
- Defining infrastructure using Kubernetes manifests
- Provisioning an AWS S3 bucket
- Verifying resources in AWS
- Cleaning up resources

## Prerequisites at a Glance

- Docker (installed and running)
- kubectl
- AWS CLI
- AWS account with S3 permissions
- 30-45 minutes

## Quick Start

1. [Read the full tutorial](index.md)
2. Follow the step-by-step instructions
3. Create your first cloud resource with Crossplane!

## What You'll Learn

By completing this tutorial, you'll understand:
- How Crossplane transforms Kubernetes into a control plane for cloud infrastructure
- How to define cloud resources using Kubernetes manifests
- How to monitor and verify resource provisioning
- Best practices for resource cleanup

## Difficulty Level

**Beginner** - No prior Crossplane experience required. Basic familiarity with Kubernetes concepts is helpful but not required.

## Estimated Time

30-45 minutes (including setup)

## Next Steps

After completing this tutorial, you'll be ready to:
- Explore more complex AWS resources
- Learn about Crossplane Compositions
- Progress to management cluster creation
- Integrate with GitOps workflows
